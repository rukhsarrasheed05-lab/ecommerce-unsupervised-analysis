# E-Commerce Unsupervised Analysis
This project analyzes customer behavior using unsupervised learning on the Online Retail dataset.

## Setup
1. Clone: `git clone https://github.com/rukhsarrasheed05-lab/ecommerce-unsupervised-analysis.git`
2. Install: `pip install -r requirements.txt`
3. Run Notebook: Open `notebook.ipynb` in Jupyter.
4. Run App: `streamlit run app.py`
5. https://ecommerce-unsupervised-analysis-9rdufp3z7yn3uadbu5vnul.streamlit.app/
